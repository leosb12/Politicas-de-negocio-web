# Shared UI

Aqui estan los componentes base reutilizables del sistema visual.

## Catalogo

- `button/` -> `app-button`
- `input/` -> `app-input`
- `select/` -> `app-select`
- `textarea/` -> `app-textarea`
- `modal/` -> `app-modal`
- `card/` -> `app-card`
- `badge/` -> `app-badge`
- `alert/` -> `app-alert`
- `table/` -> `app-table`
- `ui-toast/` -> `app-ui-toast`

## Regla de evolucion

Antes de crear HTML repetido en una feature, revisar si ya existe un primitive aqui.
Si no existe, agregarlo aqui con API simple y variantes claras.
