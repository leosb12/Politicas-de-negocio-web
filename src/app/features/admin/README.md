# Feature Admin

Este modulo concentra la administracion del sistema de politicas de negocio.

## Que contiene

- Paginas administrativas: usuarios, roles, departamentos, politicas, layout admin y canvas designer.
- Componentes internos del admin: tablas y formularios de mantenimiento.
- Servicios por dominio admin: usuarios, roles, departamentos, politicas y colaboracion en tiempo real.

## Backend relacionado

- Endpoints administrativos bajo rutas de admin/politicas.
- Integracion de colaboracion por REST + WebSocket para el editor de flujo.

## Nota de arquitectura

El admin ya esta organizado por feature y se mantiene como modulo principal de alta complejidad.
